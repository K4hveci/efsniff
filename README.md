# efsniff
Packet sniffer to work with MITM attacks

# How to Install and Use:
First clone repo:
```
git clone https://github.com/K4hveci/efsniff
```
Then move to bin:
```
cd efsniff /bin
```
To use you just need to give interface:
```
efsniff -i wlan0
```

Note: This tool uses ```scapy``` and ```scapy-http``` libs for python3.
